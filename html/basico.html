Párrafos normales

Los títulos se definen con h1 a h6, donde h1 es el más importante y h6 el menos.

La etiqueta meta charset="UTF-8" define la codificación de caracteres de la página web, asegurando que se muestren correctamente caracteres especiales y acentos.

La etiqueta meta name="viewport" content="width=device-width, initial-scale=1.0" ajusta la escala y el ancho de la página para adaptarse a dispositivos móviles, el atributo initial-scale=1.0 establece el nivel de zoom inicial cuando la página se carga. Un valor de 1.0 significa que la página se mostrará con su tamaño real, sin aplicar zoom.

La etiqueta p define párrafos de texto normales, aplicando automáticamente espacio antes y después del contenido, se utiliza cuando cuando se necesita estructurar texto normal en párrafos, ya que es semánticamente correcto y se adapta mejor a los estilos CSS. p es la etiqueta para párrafos más usada, ya que es la forma estándar de estructurar texto en páginas web.

Las pseudo clases para enlaces permiten aplicar estilos específicos según el estado del enlace. a:link se utiliza para definir el estilo de un enlace que no ha sido visitado por el usuario. a:visited se aplica a enlaces que ya han sido visitados. a:hover se activa cuando el usuario coloca el cursor sobre el enlace a:active representa el estado de un enlace en el momento en que se está haciendo clic en él.

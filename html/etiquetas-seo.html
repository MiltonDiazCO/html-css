El favicon (abreviatura de "favorite icon") es el pequeño ícono que aparece en la pestaña del navegador, en los marcadores y en la barra de direcciones cuando se visita un sitio web. Se agrega dentro del head con link rel="icon" type="image/x-icon" href="favicon.ico">

La etiqueta meta name="robots" content="index/follow" en HTML se usa para indicar a los motores de búsqueda cómo deben indexar y rastrear una página web. Se coloca dentro del head y puede contener diferentes valores, como: index/noindex→ Permite o impide que la página sea indexada. follow/nofollow → Permite o impide que los motores de búsqueda sigan o no los enlaces de la página.

La etiqueta meta name="theme-color" content="#3498db" en HTML se usa para definir el color principal de la interfaz en navegadores móviles y dispositivos compatibles. Afecta la barra de direcciones en navegadores como Chrome en Android, proporcionando una experiencia visual más personalizada.

Las Open Graph tags son metaetiquetas utilizadas en HTML para mejorar la apariencia de los enlaces compartidos en redes sociales como Facebook, LinkedIn y Twitter. Estas etiquetas definen cómo se muestra el título, la descripción, la imagen y otros elementos al compartir una página. Ejemplo: meta property="og:title" content="Título de la página"

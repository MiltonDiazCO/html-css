CSS Grid

CSS Grid es un sistema de diseño en CSS que permite crear diseños en dos dimensiones (filas y columnas) de manera precisa, flexible y responsiva.

display: grid Convierte un contenedor en un grid container, lo que permite usar todas las propiedades de Grid Layout en sus elementos hijos (los "grid items").

grid-template-columns Define el número, tamaño y comportamiento de las columnas del grid explícito. Ej: grid-template-columns: 150px 280px;

grid-template-rows Define el número, tamaño y comportamiento de las filas del grid explícito. Ej: grid-template-rows: 50px 100px 150px;

grid-explicit No es una propiedad en sí, pero se refiere al grid definido explícitamente usando grid-template-columns y grid-template-rows.

grid-implicitTampoco es una propiedad. Se refiere al grid generado automáticamente cuando hay más elementos de los que caben en el grid explícito. grid-auto-columns y grid-auto-rows

grid-auto-columns Define el tamaño de las columnas implícitas que se crean si se colocan elementos fuera del grid explícito.

grid-auto-rows Igual que el anterior, pero para filas implícitas.

grid-auto-flow Controla cómo se colocan automáticamente los elementos si no se especifican posiciones. Puede ser:

  • row (predeterminado): llenado por filas
  • column: llenado por columnas
  • dense: trata de llenar huecos

Definiciones importantes de CSS Grid

  • Contenedor: Es el elemento padre al que se le aplica display: grid; Es el que activa el sistema de cuadrícula.
  • Cuadrícula: Es la estructura de líneas (filas y columnas) que se crea dentro del contenedor. Es como una tabla invisible donde se ubican los elementos.
  • Celda: Es el espacio individual entre una fila y una columna. Como en Excel, cada punto de cruce es una celda. Los elementos pueden ocupar una o varias celdas.
  • Ítem: Es cada hijo directo del contenedor grid. Son los elementos que se posicionan en la cuadrícula.

Medidas, funciones y palabras clave

fr (fracción)

La unidad fr significa “fracción del espacio disponible” en el grid. Es muy útil para distribuir el espacio de forma proporcional. Por ejemplo, si tienes grid-template-columns: 1fr 2fr, significa que la segunda columna ocupará el doble de ancho que la primera. No se refiere a píxeles, sino a partes iguales del espacio restante después de asignar tamaños fijos (como px o auto)

auto

La palabra clave auto le dice al grid que la fila o columna debe ocupar solo el espacio necesario para su contenido. Por ejemplo, si usas grid-template-columns: auto auto, cada columna tendrá el ancho que necesite el contenido dentro de ella, sin estirarse para llenar el contenedor.

repeat()

La función repeat() es una forma más limpia de escribir varias columnas o filas con el mismo tamaño. Por ejemplo, en lugar de escribir 1fr 1fr 1fr 1fr, puedes usar repeat(4, 1fr), que crea 4 columnas iguales. También se puede usar con otros valores, como repeat(3, auto) o combinaciones más avanzadas como repeat(auto-fill, minmax(200px, 1fr)) para diseños responsivos.

span

La palabra clave span se usa en los elementos hijos del grid para indicar que deben ocupar varias celdas. Por ejemplo, si escribes grid-column: span 2, ese elemento ocupará dos columnas. Se puede usar también en filas (grid-row: span 3). Es muy útil para crear layouts más complejos donde algunos ítems ocupan más espacio que otros.

Posicionamiento con líneas negativas

Se puede hacer uso de valores negativos como -1 en propiedades como grid-column o grid-row, esto quiere decir que se esta contando desde o hasta el final de la grilla Por ejemplo, grid-column: 1 / -1; Esto significa que el elemento comienza en la primera línea de columna y se extiende hasta la última línea de columna, ocupando todas las columnas disponibles (de borde a borde).

justify-content

Esta propiedad se usa en el contenedor del grid y sirve para alinear todo el contenido del grid horizontalmente dentro del espacio disponible. Por ejemplo, si el grid no ocupa todo el ancho del contenedor, puedes moverlo hacia la izquierda, el centro o la derecha con valores como start, center, end, space-between, space-around o space-evenly.

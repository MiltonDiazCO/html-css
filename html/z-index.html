z-index controla el orden de apilamiento de los elementos en la pantalla cuando se superponen. Funciona dentro de elementos con position: relative, absolute o fixed (excepto static). Un valor mayor coloca el elemento por encima de aquellos con valores menores. Si los elementos están en el mismo contexto de apilamiento y tienen el mismo z-index, se ordenan según su posición en el código HTML (los últimos están encima).

Las pseudoclases en CSS aplican estilos a elementos en estados específicos.

Pseudo clase hover :hover se activa cuando el usuario pasa el cursor sobre un elemento.

:focus se aplica cuando un elemento, como un campo de formulario, recibe el foco.


:first-child y :last-child aplican estilos al primer y último hijo de un contenedor, respectivamente.

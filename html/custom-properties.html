Las propiedades personalizadas en CSS, también llamadas variables CSS, permiten almacenar valores reutilizables que pueden aplicarse en diferentes partes de una hoja de estilos. Se definen con el prefijo -- dentro de un selector. Las propiedades personalizadas (variables CSS) pueden tener ámbito global o local según dónde se declaren. Generalmente se definen en :root para que estén disponibles globalmente. Para el ambito local de definen dentro de un selector específico y solo afectan a sus hijos. Para utilizar una propiedad personalizada se utiliza con la función var(). Ejemplo: --bg-color: lightgray; background-color: var(--bg-color)

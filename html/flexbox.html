flexbox

Flexbox (Flexible Box Layout) es un modelo de diseño en CSS que permite organizar elementos en filas o columnas. Se utiliza para crear diseños complejos y adaptables a diferentes formatos de pantalla. Funciona con un contenedor flexible (display: flex o inline-flex) que organiza los elementos en un main axis (horizontal o vertical según flex-direction) y un cross axis (perpendicular al main axis).


flex-direction

flex-direction: Define la dirección principal en la que los elementos flexibles se organizan dentro del contenedor. Puede ser row (horizontal, de izquierda a derecha), row-reverse (horizontal, de derecha a izquierda), column (vertical, de arriba a abajo) o column-reverse (vertical, de abajo a arriba).

flex-wrap

flex-wrap: Controla si los elementos flexibles deben ajustarse en una sola línea o pueden envolver a nuevas líneas cuando no hay suficiente espacio. Puede ser nowrap (todos en una sola línea), wrap (se envuelven en múltiples líneas) o wrap-reverse (se envuelven en múltiples líneas pero en dirección inversa).

flex-flow

flex-flow: Es una propiedad abreviada que combina flex-direction y flex-wrap, permitiendo definir ambas en una sola línea. Por ejemplo, flex-flow: row wrap; organizará los elementos en fila y permitirá que se envuelvan en varias líneas si es necesario.

main axis/cross axis

main axis: Es el eje principal en el que los elementos flexibles se organizan. Su dirección depende de flex-direction. Si flex-direction es row, el main axis es horizontal; si es column, el main axis es vertical. cross axis: Es el eje perpendicular al main axis. Si flex-direction es row, el cross axis será vertical, y si flex-direction es column, el cross axis será horizontal.


justify-content

justify-content: Controla la alineación de los elementos flexibles a lo largo del main axis. Sus valores incluyen flex-start (alineados al inicio), flex-end (alineados al final), center (centrados), space-between (espaciados uniformemente con el primero y el último pegados a los extremos) space-evenly (espaciados uniformemente, incluyendo los extremos.) y space-around (espaciados uniformemente, pero con márgenes más pequeños en los extremos.).

align-content

align-content: Alinea las filas de elementos flexibles a lo largo del cross axis cuando hay varias líneas (cuando flex-wrap es diferente de nowrap). Puede ser flex-start, flex-end, center, space-between, space-around o stretch (ocupa todo el espacio disponible).

order

order: Determina el orden en que los elementos flexibles aparecen dentro del contenedor. Por defecto, todos tienen order: 0, pero se pueden asignar valores positivos o negativos para reorganizar su posición sin cambiar el HTML. Los elementos con un valor menor se mostraran primero

flex-shrink

order: Define la capacidad de un elemento flexible para reducir su tamaño cuando el espacio es limitado. Un valor mayor hace que el elemento se reduzca más en comparación con otros elementos flexibles con menor flex-shrink.

flex-grow

flex-grow: Controla cuánto puede crecer un elemento flexible en comparación con otros dentro del contenedor cuando hay espacio disponible. Un valor de 0 significa que no crecerá, mientras que un valor mayor permitirá que el elemento crezca más que los demás.
